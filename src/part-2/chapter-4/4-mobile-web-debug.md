# 移动端 Web 页面调试


### 移动 web 页面输出工具 Eruda

Eruda.js 是针对移动 web 页面调试的优秀工具。它可以直接在真机上模拟出类似 PC 上的 Chrome、Safari、FireFox 等现代浏览器的开发者调试工具，包括 Console、Elements、Network、Resources、Sources 等常见模块。
