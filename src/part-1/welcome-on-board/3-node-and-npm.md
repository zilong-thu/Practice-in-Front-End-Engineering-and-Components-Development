## 安装 Node.js 环境

### 安装 nvm

```bash
# 这个命令的意思是，从该网址下载文本内容，然后将内容交给 bash 去运行
$ curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.8/install.sh | bash
```

### 安装 Node

```bash
$ nvm install 12.0.0
```

### 安装 Yarn


```bash
$ curl --compressed -o- -L https://yarnpkg.com/install.sh | bash
```
