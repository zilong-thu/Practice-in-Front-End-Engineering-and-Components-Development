## Web 应用分类

多页面应用（Multi Page Application，MPA）是最常见的 web 形式。MPA 产品又有两大类：一类是具有复杂的数据及界面交互的胖客户端类型，例如面向企业内部的 B 端产品；另一类是对响应速度要求高、客户端轻交互、重 SSR（Server Side Render，服务器端渲染）的轻客户端类型，通常为面向大众用户的 C 端产品（尤其是移动端）。
